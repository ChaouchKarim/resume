IconifyIcon web component makes using icons as easy as icon fonts. To use "postgresql" in HTML, add this code to the document:

<iconify-icon icon="logos:postgresql"></iconify-icon>

Web component will load icon data from Iconify API and render SVG.

Make sure you import IconifyIcon web component:

<script src="https://code.iconify.design/iconify-icon/1.0.3/iconify-icon.min.js"></script>

Web component works with almost all modern UI frameworks. Click documentation link below.
